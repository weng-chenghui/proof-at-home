User Guide

Learn by creating: install the tools, explore content, then build your own.

1

Install the CLI

Run the one-line installer to download a pre-built binary:

curl -fsSL https://pah.fly.dev/install.sh | sh

The installer places pah in your PATH. To build from source instead: git clone https://github.com/pah-org/proof-at-home.git && cargo build --release

2

Install a proof assistant

You need at least one proof compiler to verify proofs:

  • Rocq for Rocq/Coq proofs (~10 min with opam)
  • Lean 4 for Lean proofs (~5 min with elan)

Check what's detected: pah tools check

3

Browse existing content

Explore what's already been formalized:

pah series list
pah lesson list

Or browse online at the Series & Lessons page. The Formalizing Finance series is a good place to start.

4

Pick a topic to formalize

Choose any domain you're curious about — finance, physics, music theory, game theory, anything. Browse existing conjectures or define your own:

pah conjecture list

See all available conjectures on the Conjectures page.

5

Prove conjectures

Three methods, all equally credited:

AI-assisted — LLM generates proofs, compiler verifies:

pah setting set          # Set AI provider and API key
pah contribution run

Pair with coding assistant — export, solve with Claude Code / Cursor / Copilot / Windsurf, submit:

pah conjecture export prob_001 | pbcopy
# ... prove with your AI assistant ...
pah proof submit prob_001 proof.lean --method pair-proved

Manual — write proofs yourself:

pah proof submit prob_001 ./my_proof.v

The CLI compiles proofs with rocq c or lean to verify correctness, submits the results, and archives everything with a SHA-256 hash.

6

Create lessons and series

Turn your proofs into structured educational content:

pah lesson create --topic <t> --conjectures id1,id2
pah series create --lessons id1,id2

AI annotations are automatically generated to help future learners. See the Content Pipeline guide for details.

7

View your contributions

See your contributions and their status:

pah contribution list

Every contribution earns a permanent credit record. Browse all contributions and their verification hashes in the gallery.

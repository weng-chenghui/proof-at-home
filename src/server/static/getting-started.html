Getting Started

Go from zero to your first verified proof contribution in seven steps.

1

Prerequisites

Make sure you have the following installed:

  • Rust (1.70+) — needed to build the CLI
  • An Anthropic API key — powers the AI prover
  • Rocq for Rocq proofs, or Lean 4 for Lean proofs
2

Install the CLI

Build from source:

git clone https://github.com/pah-org/proof-at-home.git
cd proof-at-home
cargo build --release

The binary will be at target/release/proof-at-home. Add it to your PATH or run it directly.

3

Configure

Run the interactive setup wizard:

./target/release/proof-at-home init

This will ask for your name, username, Anthropic API key, and the server URL. Config is saved to ~/.proof-at-home/config.toml.

4

Set your donation budget

Choose how much API credit to donate per run:

./target/release/proof-at-home donate

Read and accept the legal agreement, then pick an amount ($1 -- $10 or custom). This caps the API cost for a single contribution run.

5

Submit your first certificate

Start proving open conjectures:

./target/release/proof-at-home prove

The CLI fetches conjectures from the server, calls Claude to generate proofs, verifies each with rocq c or lean, submits certificates, and archives everything with a SHA-256 hash.

6

Check your contribution

See your stats and contribution history:

./target/release/proof-at-home status

This shows your total certificates, budget spent, and contribution IDs.

7

View in the NFT Gallery

Every contribution generates NFT-compatible metadata. Browse all contributions and their verification hashes in the gallery.

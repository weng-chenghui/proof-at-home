Pair Prover Guide

Prove theorems interactively with your AI coding assistant.

No API key or budget configuration needed. You use your existing AI subscription (Claude Code, Cursor, Copilot, Windsurf, Cline, Aider). Proof@Home only needs the finished proof — verification runs locally on your machine.

1

Install

Install the CLI and verify your setup:

curl -fsSL https://pah.fly.dev/install.sh | sh
pah tools check
2

Sign up & log in

Create an account on the web UI, then authenticate the CLI:

pah auth login
3

Browse conjectures

Find an open conjecture to work on:

pah conjecture list

Or browse them on the web UI.

4

Export a conjecture

Export the conjecture as a prompt for your AI assistant:

pah conjecture export prob_001

Copy to clipboard (macOS):

pah conjecture export prob_001 | pbcopy

Other formats: --format json for structured data, --format source for a raw compilable file.

5

Prove with your AI assistant

Paste the exported prompt into your AI coding assistant — Claude Code, Cursor, Copilot, Windsurf, Cline, or Aider. Iterate until the proof compiles. Save the result as a .lean or .v file.

6

Submit your proof

Submit from a file:

pah proof submit prob_001 proof.lean --method pair-proved

Or pipe from clipboard (macOS):

pbpaste | pah proof submit prob_001 --stdin --method pair-proved

The CLI verifies the proof locally with the proof assistant compiler, then submits it to the server.

7

Create expositions too

Pair provers can also create rich expositions with text, math, and visualizations:

# Export the exposition prompt
pah exposition export --for=prob_001 | pbcopy

# Paste into your AI assistant, iterate until the JSON output is correct...

# Submit the result
pbpaste | pah exposition create --for=prob_001 --stdin --method pair-proved

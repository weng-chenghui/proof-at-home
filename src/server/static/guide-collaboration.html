Collaborator Quick Start

Fix, improve, or extend others' work. Clone the shared pool, browse contributions, and seal your own version.

1

Install

Install the CLI in one command. The installer optionally sets up proof toolchains (Lean 4, Rocq):

curl -fsSL https://pah.fly.dev/install.sh | sh

Run pah tools check any time to verify your setup.

2

Sign up & log in

Create an account on the web UI, then authenticate the CLI:

pah auth login
3

Clone the pool

Clone the shared data repository to your machine:

pah pool clone

By default clones to ~/.proof-at-home/pool/. Use --dir <path> for a custom location.

4

Pull latest work

Fetch and rebase to get the latest contributions:

pah pool pull
5

Browse & improve

Check the pool status, browse contributions, and fix or improve a proof:

pah pool status

# Browse the pool directory
ls ~/.proof-at-home/pool/contributions/

# Edit a proof file to fix or improve it
6

Submit & seal

Submit your improved proof and seal it with NFT metadata:

pah proof submit <conjecture_id> <proof_file>
pah contribution seal <id>

Your NFT will include a "Based On" attribute linking to the original contribution.

7

Push back

Share your improvements with the community:

pah pool push

What pah helps with

  • Text-based artifacts: proofs (.v, .lean), conjectures (JSON), expositions
  • Git sync: clone, pull, push the shared pool with one command
  • Identity auto-config: git user.name/email set from your Proof@Home identity
  • NFT sealing with provenance: your work is credited, with a link to the original

What pah does NOT help with

  • Merge conflicts — if a rebase conflicts, use standard git commands (git rebase --continue / --abort)
  • Video, images, or binaries — the pool is for text artifacts; host media on YouTube, IPFS, etc.
  • Real-time editing — collaboration is async via git; there is no live co-editing
  • Modifying others' sealed NFTs — sealed contributions are immutable; you create your own version instead

Collaboration flow

Alice (prover)                         Bob (collaborator)
    |                                       |
    | pah contribution run                  |
    | pah contribution seal <id>            |
    | → Alice's NFT sealed                  |
    |                                       |
    |                                  pah pool clone
    |                                  pah pool pull
    |                                       |
    |                                  # browse pool/contributions/
    |                                  # fix Alice's proof
    |                                       |
    |                                  pah proof submit <conj> fix.v
    |                                  pah contribution seal <bob-id>
    |                                  → Bob's NFT (Based On: Alice)
    |                                       |
    |                                  pah pool push
    |                                       |
    | pah pool pull                         |
    | # sees Bob's improvement              |

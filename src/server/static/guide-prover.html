Prover Quick Start

Prove theorems with AI, by hand, or both.

1

Install

Install the CLI in one command. The installer optionally sets up proof toolchains (Lean 4, Rocq):

curl -fsSL https://pah.fly.dev/install.sh | sh

Run pah tools check any time to verify your setup.

2

Sign up & log in

Create an account on the web UI, then authenticate the CLI:

pah auth login
3

Configure AI provider (optional — only for AI-assisted mode)

Set up your AI provider (Anthropic, OpenAI, or Ollama) and budget. Skip this step if you plan to write proofs by hand.

pah setting set provider
pah setting set api_key
pah setting set budget

Use pah provider list to see available providers. Ollama runs locally for free.

4

Browse conjectures

Find an open conjecture to work on:

pah conjecture list

Or browse them on the web UI.

5

Prove

AI-assisted — fetch conjectures, AI-prove them, verify, and seal:

pah contribution run

Manual — write a .v (Rocq) or .lean (Lean) file, then submit:

pah proof submit <conjecture_id> <proof_file>

# Or batch-submit a directory:
pah proof submit --dir ./proofs/
6

View your NFT credit

Your submission is sealed with NFT metadata. Browse the gallery to see it.

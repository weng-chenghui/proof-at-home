Expositor Quick Start

Create rich expositions with text, math, and interactive visualizations. Works with your AI API key or coding assistant.

1

Install

Install the CLI in one command:

curl -fsSL https://pah.fly.dev/install.sh | sh
2

Sign up & log in

Create an account on the web UI, then authenticate the CLI:

pah auth login
3

Configure AI provider

An AI provider is needed to generate expositions. Set up your provider and API key:

pah setting set provider
pah setting set api_key

Supports Anthropic Claude, OpenAI ChatGPT, and Ollama (free local models). Run pah provider list for details.

Using Claude Code, Cursor, or another AI coding assistant? Skip API key setup — see "Pair Prover mode" in step 5 below.

4

Browse resources

Find conjectures, contributions, or certificates to create expositions for:

pah conjecture list
pah contribution list
Browse Conjectures
5

Create an exposition

API-assisted mode (uses your API key):

# For a conjecture
pah exposition create --for=prob_001

# For a contribution
pah exposition create --for=contrib_abc

# For a certificate
pah exposition create --for=cert_xyz

Pair Prover mode (uses your AI coding assistant):

# Export the prompt for your AI assistant
pah exposition export --for=prob_001 | pbcopy

# Work with your AI assistant to create the exposition...
# Then submit the result:
pbpaste | pah exposition create --for=prob_001 --stdin --method pair-proved
6

View your NFT credit

Expositions generate NFT metadata just like contributions and certificates. Check the gallery to see your work.

Expositor Quick Start

Use AI to explain proved theorems in plain language. Turn formal proofs into human-readable narratives.

1

Install

Install the CLI in one command:

curl -fsSL https://pah.fly.dev/install.sh | sh
2

Sign up & log in

Create an account on the web UI, then authenticate the CLI:

pah auth login
3

Set your API key

An Anthropic API key is needed to generate explanations with Claude:

pah setting set anthropic_api_key
4

Browse proved conjectures

Find conjectures that have been proved and are ready to be explained:

pah conjecture list
Browse Conjectures
5

Parse a proof

Generate a human-readable explanation from a local proof file or from a server contribution:

# From a local file
pah proof parse my_proof.v

# From a server contribution
pah proof parse --contribution <id> --conjecture <id>

# With a custom strategy
pah proof parse my_proof.lean --by my-custom-parse
6

View your NFT credit

Expositions generate NFT metadata just like contributions and certificates. Check the gallery to see your work.

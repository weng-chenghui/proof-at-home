Craftsman Quick Start

Write formal proofs by hand in Rocq or Lean.

1

Install

Install the CLI in one command. The installer optionally sets up proof toolchains (Lean 4, Rocq):

curl -fsSL https://pah.fly.dev/install.sh | sh

Run pah tools check to verify Rocq or Lean is available.

2

Sign up & log in

Create an account on the web UI, then authenticate the CLI:

pah auth login
3

Browse conjectures

Find an open conjecture to work on:

pah conjecture list

Or browse them on the web UI.

4

Write your proof

Write a .v (Rocq) or .lean (Lean) file proving the conjecture. Use your favourite editor and proof assistant.

5

Submit

Submit a single proof or a whole directory. The CLI verifies, then seals with NFT metadata automatically:

pah proof submit <conjecture_id> <proof_file>

# Or batch-submit a directory:
pah proof submit --dir ./proofs/
6

View your NFT credit

Your submission is sealed with proof_mode "manual". Browse the gallery to see it.

Contributor Quick Start

Donate spare AI compute to prove theorems automatically.

1

Install

Install the CLI in one command:

curl -fsSL https://pah.fly.dev/install.sh | sh
2

Sign up & log in

Create an account on the web UI, then authenticate the CLI:

pah auth login
3

Configure AI provider

Set up your AI provider (Anthropic, OpenAI, or Ollama). The interactive wizard will guide you:

pah setting set provider
pah setting set api_key

Use pah provider list to see available providers. Ollama runs locally for free.

4

Set budget

Cap how much API credit to spend per run:

pah setting set budget
5

Prove!

Fetch conjectures, AI-prove them, verify, and seal with NFT metadata:

pah contribution run
6

View your NFT credit

Check nft_metadata.json in the contribution directory, or browse the gallery.

Certifier Quick Start

Review and certify proof packages submitted by others.

1

Install

Install the CLI in one command:

curl -fsSL https://pah.fly.dev/install.sh | sh
2

Sign up & log in

Create an account on the web UI, then authenticate the CLI:

pah auth login
3

Configure AI provider

Set up your AI provider for proof comparison:

pah setting set provider
pah setting set api_key

Supports Anthropic Claude, OpenAI ChatGPT, and Ollama (free local models).

4

Create session

Start a certification session — this fetches proof packages from the server:

pah certificate create
5

Compare proofs

Use AI to compare proofs across packages:

pah certificate compare
6

Generate & edit report

Generate a TOML report template (opens in $EDITOR if set). Edit your rankings and notes, then re-run to validate:

pah certificate report
7

Seal & publish

Seal your certification with NFT metadata, then publish it:

pah certificate seal
pah certificate publish <id>
8

View your NFT credit

Check certification_nft_metadata.json, or browse the gallery.
